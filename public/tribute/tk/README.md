# TK Tribute — assets

All assets for the TKF Tribute (`/tk-tribute.html`) live here and are served at
`/tribute/tk/…` (Vite copies `public/` to the site root at build time).

## Current

- `portrait.webp` — archival side-profile portrait of TK. Used as the Room 03
  ("The Person & The Voice") archival photograph.

## Expected / forthcoming (drop the real files in with these exact names)

- `letter-to-tk.mp3` — Luscious Honey's recorded letter to TK (Room 02).
  The Room 02 card is an intentionally inactive fixture until this file exists;
  no audio is wired yet.
- `portrait-large.webp` — optional higher-resolution portrait, only if a larger
  source is needed for the click-to-enlarge view. Not currently referenced.
- Future gallery images — additional Tribute artwork, named descriptively
  (e.g. `gallery-01.webp`).

When adding a real asset, keep the filename above (or update the matching path
in `tk-tribute.html`). Do not commit empty/placeholder binaries — reference a
path only once the real file is present.
